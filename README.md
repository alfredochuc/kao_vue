## Configuration
Create a new file named `.env` on the root of your project with your VUE_APP_SPOTIFY_CLIENT_ID and VUE_APP_SPOTIFY_CLIENT_SECRET

```
VUE_APP_SPOTIFY_CLIENT_ID=<key>
VUE_APP_SPOTIFY_CLIENT_SECRET=<key>
```
